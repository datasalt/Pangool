Custom Serialization

You can use your own custom stateful serializers and deserializers for serializing OBJECT fields implementing FieldSerializer and FieldDeserializaer interfaces. 

    public static interface FieldSerializer<T> extends Serializer<T>{
      public void setProps(Map<String,String> properties);
    }

    public static interface FieldDeserializer<T> extends Deserializer<T>{
      public void setProps(Map<String,String> properties);
    }

As example, Pangool offers stateful Avro stateful serialization. This means that you can use any Avro data type for one of yours Pangool fields. 

  static org.apache.avro.Schema getAvroSchema(){
    return org.apache.avro.Schema.createArray(org.apache.avro.Schema.create(Type.INT));
  }
